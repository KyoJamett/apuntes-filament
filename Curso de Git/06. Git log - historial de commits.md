El comando `git log` se usa para mostrar todos los commits de un repositorio.

```
git log
```

Por defecto, este comando muestra, por cada commit:
- el SHA
- el autor
- la fecha
- y el mensaje
esto se muestra por defecto, por lo que podría mostrar más.

### Comandos en git log
- scroll con mouse, o flechas arriba o abajo
- para salir, usa la tecla Q

### Git log --oneline
Si no necesitas ver tanta info sobre los commits y deseas ver, por ejemplo, el nombre y el id de los commits, puedes usar el comando:

```
git log --oneline
```

![[Pasted image 20260211121520.png]]
 _En este caso, se visualiza los primeros 7 caracteres del SHA y el nombre del commit_


### Git log --stat
Con este flag puedes ver el detalle de los archivos editados por commit

![[Pasted image 20260211122049.png]]

### Ir a un commit específico
Puedes revisar el detalle de un commit específico con `git show` y el SHA del commit:

```
git show fdf5493
```
