```
git clone <path-to-repository-to-clone> <dir-name>
```

El comando `git clone` permite clonar un repositorio, para esto, luego del comando se agrega la url del repositorio remoto donde se aloja el proyecto.

Este comando:
- Toma el path de un repo existente
- Por defecto crea un directorio con el mismo nombre que el repositorio que será clonado
- Se le puede pasar un segundo parámetro para cambiar el nombre del directorio donde se va a guardar el proyecto
